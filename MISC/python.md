# python开发
## 环境问题
### 各种环境工具
- virtualenv 所要解决的是同一个库不同版本共存的兼容问题，太老了
- pyvenv 和virtualenv一样但是只支持python3.3及更高版本
- venv `python -m venv`等价于pyvenv
- pyenv 管理一台机器上的多个 Python 版本(包含virtualenv后可以顺便管理库版本)
